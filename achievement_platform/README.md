Шаг 1. Подготовка окружения
Установи Python (если еще нет).
Создай папку для проекта, например achievement_platform.
Открой терминал (командную строку) в этой папке и установи нужные библиотеки:

bash

pip install streamlit pandas

streamlit — для интерфейса.
pandas — для удобной работы с таблицами (отчеты).


Шаг 2. Запуск демонстрации
В терминале в папке с проектом напиши:

bash

streamlit run app.py

Автоматически откроется браузер с адресом http://localhost:8501.
Готово! У тебя есть работающее веб-приложение.
